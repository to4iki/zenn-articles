HHKBはFnキーとの組み合わせで矢印移動を実現する。

- 上: fn + [
- 下: fn + /
- 左: fn + ;
- 右: fn + '

## Karbinerをインストールする
MacOSのキーバインドをカスタマイズするため [Karabiner\-Elements](https://karabiner-elements.pqrs.org/) をインストールする。

## Happy Hacking Keyboard Compatible Mode (rev 2)をインポートする
HHKB用のキー配列を実現する[ルール](https://ke-complex-modifications.pqrs.org/?q=Happy%20Hacking%20Keyboard%20Compatible%20Mode)を追加する。
